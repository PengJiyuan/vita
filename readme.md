----------

###简介：

> 响应式设计，PC端、手机端均可完美运行

> 支持触屏翻页（整屏翻页），动画可重复触发

> 因图片过多，图片未加载会影响动画整体效果，所以图片全加载完成才显示页面

> 改进：采用gulp + sass 编辑，scss代码嵌套清晰，修改维护更为便捷。
